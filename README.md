# Peta-Pengklasifikasian-Rasio-Jenis-Kelamin-Kec.GunungKidul-Tahun-2024
Pada Peta Pengklasifikasian Rasio Jenis Kelamin Kec.GunungKidul Tahun 2024 ini terdata 11 Kecamatan di daerah Gunung Kidul dengan pesebaran presentase dan jumlah jenis kelamin baik perempuan maupun laki-laki kisaran 94-100 orang.
